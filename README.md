# mern-daily-practice

Daily MERN stack practice — one concept per day covering React, Node.js, Express, and MongoDB. Built during MCA to prep for placements.

---

## 📁 Structure

```
mern-daily-practice/
├── react/
├── node-express/
└── mongodb/
```

Each day folder contains the working code and a `notes.md` with key takeaways.
